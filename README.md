# Aarun's Modern Portfolio

A beautifully designed, modern portfolio website inspired by Apple's design philosophy. Showcasing technology engineering expertise, projects, and professional background.

## Features

✨ **Modern Design**
- Clean, minimalist Apple-inspired interface
- Smooth animations and transitions
- Dark mode support (automatic based on system preferences)

📱 **Responsive & Mobile-First**
- Fully responsive design
- Works perfectly on desktop, tablet, and mobile
- Touch-friendly interactive elements

⚡ **Performance Optimized**
- Lightweight and fast loading
- No external dependencies required
- Semantic HTML for better SEO

🎯 **Key Sections**
- **Hero**: Compelling introduction
- **About**: Overview of expertise and skills
- **Projects**: Showcase of key technologies and achievements
- **Experience**: Professional journey timeline
- **Stats**: Key metrics and accomplishments
- **Contact**: Multiple ways to get in touch

## Files

- `index.html` - Main portfolio page with semantic HTML structure
- `styles.css` - Complete styling with responsive design and animations
- `script.js` - Interactive features and smooth scrolling
- `README.md` - This file

## Deployment to GitHub Pages

### Quick Start

1. Replace the contents of your `send2aarun.github.io` repository with these files
2. Push to GitHub:
   ```bash
   git add .
   git commit -m "Update portfolio with modern design"
   git push origin main
   ```

3. Your portfolio will be live at: `https://send2aarun.github.io/`

### Development Locally

You can preview the portfolio locally by:

**Option 1: Python**
```bash
python -m http.server 8000
```

**Option 2: Node.js (with http-server)**
```bash
npx http-server
```

Then open `http://localhost:8000` in your browser.

## Customization Guide

### Update Your Information

Edit `index.html` to customize:
- Your name and title
- Bio and description
- Project descriptions
- Contact information
- Social media links

### Change Colors

In `styles.css`, modify the CSS variables at the top:
```css
:root {
    --accent-color: #0071e3;  /* Blue accent */
    --text-color: #1d1d1d;     /* Dark text */
    --secondary-color: #f5f5f7; /* Light background */
}
```

### Add Projects

Add new project cards in the projects section:
```html
<article class="project-card">
    <div class="project-header">
        <h3>Your Project Name</h3>
        <span class="project-tag">Category</span>
    </div>
    <p class="project-description">Description...</p>
    <ul class="project-tech">
        <li>Tech 1</li>
        <li>Tech 2</li>
    </ul>
    <a href="#" class="project-link">View Details →</a>
</article>
```
