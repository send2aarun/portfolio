# Publishing Your Portfolio to GitHub Pages

## Quick Start (3 Steps)

### Step 1: Initialize Git & Add Remote
Open Terminal and run:
```bash
cd /Users/aarun/Desktop/aarunProject
git init
git remote add origin https://github.com/send2aarun/send2aarun.github.io.git
```

### Step 2: Add Files & Commit
```bash
git add .
git commit -m "Launch redesigned portfolio - modern, responsive design"
```

### Step 3: Push to GitHub
```bash
git branch -M main
git push -u origin main
```

---

## Alternative: UI Method (If Git Issues)

If you encounter git errors, use GitHub Desktop or the GitHub web interface:

1. **Go to GitHub**: https://github.com/send2aarun/send2aarun.github.io
2. **Upload files directly**:
   - Click "Add file" → "Upload files"
   - Select all files from `/Users/aarun/Desktop/aarunProject/`
   - Click "Commit changes"

---

## Verify Deployment

Once pushed, your portfolio will be live at:
